# BrainBox AI

BrainBox AI is a smart personal memory assistant built with Flutter. It helps you organize notes, images, voice recordings, and PDFs in one private and offline-friendly workspace.

## ✨ Features

- Create and manage notes with tags
- Save and view images with descriptions
- Record and play voice notes
- Store and open PDF files
- Search across your content intelligently
- Ask BrainBox natural-language questions such as:
  - “Where did I save my papers?”
  - “Show me my work notes”
  - “Find something related to money”
- Dark and light mode support
- Local storage with Hive for offline use

## 🧠 What makes it special

BrainBox AI is designed to feel like a personal digital memory center. Instead of scattering your ideas across different apps, everything is kept in one place with a clean, modern experience.

## 🛠️ Tech Stack

- Flutter
- Provider for state management
- Hive for local storage
- Google Fonts for typography
- Image Picker, File Picker, Audio Recorder, PDF Viewer, and Permission Handler

## ▶️ Getting Started

### Prerequisites

Make sure you have Flutter installed on your machine.

- Flutter SDK 3.11.4 or newer
- Android Studio / VS Code
- An emulator or physical device

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

## 📁 Project Structure

- lib/main.dart — Application entry point
- lib/screens/ — Main app screens
- lib/providers/ — App state and data providers
- lib/services/ — Business logic and search features
- lib/models/ — Data models
- lib/widgets/ — Reusable UI components
