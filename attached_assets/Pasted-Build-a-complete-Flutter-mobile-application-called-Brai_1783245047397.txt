Build a complete Flutter mobile application called "BrainBox AI".

The app is a smart personal memory assistant that stores notes, images, voice notes, and PDFs locally and allows intelligent search without using any paid AI APIs.

---

## 🔥 TECH STACK REQUIREMENTS

- Flutter (latest stable)
- State Management: Provider
- Local Database: Hive (mandatory)
- No backend servers
- Fully offline-first application
- Clean Architecture (recommended but not strict)
- Null safety enabled

---

## 🌍 LANGUAGE SUPPORT

The app must support:
- Arabic (RTL support fully implemented)
- English (LTR support)

Include a language switcher in settings.

---

## 📱 RESPONSIVENESS

- Fully responsive UI for:
  - Small phones
  - Large phones
  - Tablets
- Adaptive layouts using MediaQuery or LayoutBuilder
- Proper spacing and scaling

---

## 🎨 UI/UX REQUIREMENTS

The design must be:
- Modern
- Minimal
- Elegant
- Smooth animations (optional but preferred)
- Inspired by apps like Notion / Google Keep / Apple Notes

Use:
- Cards
- Soft shadows
- Rounded corners
- Clean typography

Dark mode support is optional but preferred.

---

## 🧠 CORE FEATURES (MVP – FREE VERSION)

### 1. Notes Module
- Create / edit / delete notes
- Store in Hive
- Add tags
- Search notes locally

---

### 2. Image Memory
- Save images locally
- Add description manually
- Store metadata in Hive

Optional:
- OCR using Google ML Kit (free) to extract text from images

---

### 3. Voice Notes
- Record audio notes
- Save locally
- Optional: speech-to-text using device capabilities (no paid APIs)

---

### 4. PDF Storage
- Upload and store PDFs locally
- View PDFs inside app
- Save metadata in Hive

---

### 5. Smart Search (NO AI APIs)
Implement a smart local search system:
- Keyword-based search
- Tag-based filtering
- Simple rule-based categorization
- Search across all modules (notes, images, PDFs, voice notes metadata)

---

### 6. Timeline View
- Show all user data in chronological order
- Group by date
- Include notes, images, and voice entries

---

### 7. Ask BrainBox (FREE VERSION – NO AI)
This feature should NOT use any external AI API.

Instead:
- Implement local search + keyword matching
- Return best matching stored results
- Show “smart-like” responses based on indexed data

Example:
User asks: "Where did I save my car papers?"
App searches Hive data and returns matching results.

---

## 🗂️ DATA STRUCTURE (HIVE MODELS)

Create Hive models for:
- NoteModel
- ImageModel
- VoiceNoteModel
- PdfModel
- TagModel (optional)

Each model must include:
- id
- title
- content/path
- createdAt
- tags (if applicable)
